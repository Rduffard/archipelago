# Pairing Matrix

Archipelago can treat the six core attributes as a shared language. Instead of inventing a disconnected skill list for every part of play, we can derive named pairings from two attributes and reuse them across social, combat, exploration, arcane, and relic systems.

## Design Rule

Every pairing should answer one question:

What kind of action happens when these two parts of a person meet?

That keeps the names grounded in fiction instead of drifting into generic RPG taxonomy.

## Full Matrix

- `Might + Agility = Skirmish`
  Fast violence, dueling tempo, boarding actions, closing the distance.
- `Might + Wit = Leverage`
  Strength used intelligently: breaching, hauling, force applied with engineering sense.
- `Might + Spirit = Conviction`
  Zeal, soul-force, prophetic presence, a declaration that lands like a blow.
- `Might + Resolve = Pressure`
  Command, intimidation, threat, social force backed by will.
- `Might + Instinct = Pursuit`
  Hunting, running prey down, predatory momentum, relentless survival pressure.
- `Agility + Wit = Precision`
  Aim, exact timing, surgical action, control under pressure.
- `Agility + Spirit = Flourish`
  Performance, charisma through motion, ritual display, elegant magnetism.
- `Agility + Resolve = Balance`
  Footwork, storm-deck stability, moving well when the world pitches around you.
- `Agility + Instinct = Reflex`
  Split-second reaction, evasive movement, quickdraw instinct, live danger response.
- `Wit + Spirit = Guile`
  Deception, misdirection, veiled intent, layered conversation.
- `Wit + Resolve = Tactics`
  Discipline, planning, coordination, smart decisions that hold under stress.
- `Wit + Instinct = Sense`
  Reading danger, noticing patterns, scouting, navigation, situational understanding.
- `Spirit + Resolve = Grace`
  Diplomacy, poise, calm authority, trust-building, social steadiness.
- `Spirit + Instinct = Attunement`
  Omen-reading, spirit sensitivity, relic resonance, feeling the unseen.
- `Resolve + Instinct = Nerve`
  Courage, survival calm, fear resistance, acting while panic is still forming.

## Sheet Recommendation

Not every pairing should always live on the default character sheet.

Recommended always-visible social trio:

- `Grace`
- `Guile`
- `Pressure`

Recommended next layer once the sheet grows:

- `Precision`
- `Sense`
- `Attunement`
- `Conviction`
- `Skirmish`

That gives the system depth without overwhelming players with fifteen visible numbers at once.

## Why This Works

- It keeps all subsystems speaking the same language.
- It makes new mechanics easier to invent later.
- It gives relics, callings, and reputation a clean place to hook in.
- It feels bespoke to Archipelago instead of inherited from another game.

## Example Uses

- `Grace` for persuasion, diplomacy, etiquette, calming a room.
- `Guile` for lies, false identities, hidden motives, social traps.
- `Pressure` for hard bargains, threats, command presence, making someone fold.
- `Attunement` for relic reading, corruption sensing, spirit contact, occult navigation.
- `Sense` for scouting, ambush detection, danger reads, environmental clues.
- `Precision` for firearms, sabotage, surgery, exact action under time pressure.

## Future Rule Pattern

When the game needs a roll, it can ask for:

- a base attribute
- a named pairing
- or a full sheet stat like `Guard` or `Focus`

That gives Archipelago three layers of resolution:

- attributes for raw identity
- pairings for action style
- derived stats for defenses and persistent ratings
